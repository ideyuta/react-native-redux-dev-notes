Development Rules and Knowledge for React Native + Redux


## Installation

```
git clonet https://github.com/tongariboyz/react-native-redux-dev-notes.git
npm install
```


## Development

### サーバ起動

```
npm start
```

http://localhost:4000 にアクセスすることでプレビューできる。


### ディレクトリ構成

```
.
├── README
├── SUMMARY.md <= 目次
├── ja/ <= .mdの章を追加する
├── ja/resources/ <= 画像などリソースファイルを追加する
├── prh.yml
├── src/ <= サンプルコード
└── test/ <= サンプルコードのテスト
```


## Build

### HTML

```
npm run build
```
